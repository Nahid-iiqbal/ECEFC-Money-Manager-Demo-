# Exports Directory

⚠️ **LOCAL ONLY** - This folder is gitignored and should never be committed.

## Purpose

This directory contains anonymized analytics exports generated by the `tools/export_anonymized_analytics.py` script.

## Privacy Guarantees

All exports in this folder adhere to strict privacy rules:

- ✅ Aggregate counts (total users, expense counts)
- ✅ Category totals across all users (no user breakdown)
- ✅ Monthly buckets (YYYY-MM format only)
- ✅ Type distributions
- ❌ NO user_id, email, username, profile_name
- ❌ NO raw expense descriptions or names
- ❌ NO exact timestamps (day/hour/minute)
- ❌ NO individual user data

## Usage

Generate a new export:

```bash
python tools/export_anonymized_analytics.py
```

Output files are named: `analytics_YYYYMMDD_HHMMSS.json`

## Sample Output Structure

```json
{
  "export_metadata": {
    "exported_at": "2025-01-15T10:30:00",
    "version": "1.0.0",
    "privacy_level": "anonymized_aggregates_only"
  },
  "user_statistics": {
    "total_users": 150,
    "users_with_profile": 120
  },
  "expense_analytics": {
    "total_expenses": 5000,
    "category_breakdown": {
      "Food": {"total_amount": 25000.00, "count": 1500}
    },
    "monthly_breakdown": {
      "2025-01": {"total_amount": 8000.00, "count": 400}
    }
  }
}
```

## Important

Never commit these files. If you see export files appearing in git status, ensure `.gitignore` includes `exports/`.
